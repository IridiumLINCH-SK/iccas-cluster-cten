The original code was written by Zi-yue Wang, whose modification was made by Ning-zheng Li (Iridium LINCH-SK).

The CTEN model has following requirements: 
- Python (>= 3.9)
- NumPy
- pandas
- scikit-learn
- SciPy
- PyTorch
- matplotlib

The label used in the training is the opposite number of atomization energy per atom (AEPA), which is forming energy per atom (FEPA).
